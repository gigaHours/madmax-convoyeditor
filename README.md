# madmax-convoyeditor

Opens global/convoys.bl/global/convoys.xml (button or drag & drop)
Parses all 14 convoys, routes, waypoints
Draws routes in 2D (X/Z projection) with colors by territory (Jeet — blue, Gutgash — orange, Pink Eye — purple, MM3030 — red)
Saves the modified XML back

Editing points:

Dragging points with the mouse
Adding new points (using the "+" tool or right click → Insert before/after)
Deleting points (Delete/Backspace or right click → Delete)
Duplicating points
Editing X/Y/Z coordinates in the Properties panel

Navigation:

Mouse wheel — zoom
Dragging an empty space or Space+drag — pan
F — enter everything on the screen
The minimap is in the lower right corner

Background map:

"Load Map" button — load the map image
Shift+drag — move the map
Shift+wheel — zoom the map

Other:

Undo/Redo (Ctrl+Z / Ctrl+Y)
Arrows ←/→ to switch between points
Visibility of convoys in the list (●/O button)
